# The WebApp_AWS

Web App for testing Students Reading Level by using AWS Transcribe. <br>

Web App deploy on the Heroku <br>
Web Link: https://pennez-app.herokuapp.com/


## Back end

### Tech used

* express.js
* Node.js
* MongoDB Atlas
* AWS sdk
* Passport JS

### Getting started

* `npm run dev`
